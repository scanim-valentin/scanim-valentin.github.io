Software Development

Decentralized chat agent The goal of this app is to allow users on a same network to communicate in a decentralized fashion while enabling the possibility to access their conversation history.
This project was made in Java. It involves :
  • Extensive UML designs
  • Socket programming
  • SQL querying
Cipherdit My contribution to this experimental work was part of a larger "initiation to research" ambitious school project. It aimed at creating a privacy focused "level-2" online social network Android app in Java. It involves exchanging links to message content stored and encrypted on a private Dropbox cloud storage over an already existing social network (Reddit) by using steganography.

My team was in charge of implementing the encryption layer between the user and cloud storage.
Ford-Fulkerson implementation in OCaml Implementation of the Ford-Fulkerson maximum-flow algorithm in OCaml, a functional programming language.
Real-time wheeled robot control Remote controls for a small wireless robot in C++. The main challenge was to manage real time concurrent programming and priorities.

Embedded Development

Sailing Boat Replica A set of bare metal STM23 libraries written in C to automate the direction and sail deployment of a sailing boat replica. This project was made using ARM's Keil.
Laser game This work was made in assembly language with Cortex M3 instruction set via ARM's Keil and runs on STM32. The goal was to implement real time FFT for laser detection of specific wavelengths and PWM for playing sound.
BuzzQuizz A C script that runs on Arduino Uno to control RGB LED matrix displays and buzzers with a set of quiz buttons.
Perceptron puzzle This project is a C script that runs on a Teensy Cortex-M based board. It is part of a physical electronic game set made to learn basic AI concepts.
The goal is to find the correct parameters, symbolized by the LED rings changing their respective colors as the player turns each rotary encoders. Let these parameters be found, the device manages to categorize each element of a set of labeled card containing RFID chips.
Closest Neighbours puzzle This small interactive puzzle written in Python to run on a Raspberry Pi is part of an educative escape game meant to teach about AI.
It uses BeautifulSoup to convert SVG content to set of labeled dots of different sizes and colors and TKinter to represent and link these dots by clicking on them.
It is a puzzle, so I will not elaborate on what the expected result is (not because I completely forgot about it).

Game Programming

INSArio: Super PPA A 2D mario-themed platformer game made in Unity with my school's IT club.
I was in charge of implementing random terrain generation based on adjacency constraints as well as the parallax background.
Marre des canards (INSA jam) Quite a strange 2D fight game made in Unity as a submission to a school game jam.
The goal is to steal the opponent's bread crumbs.
I was in charge of implementing collision, movement and HUD.
Jam Theme: "Marre des canards" ( a word play between "Duck pond" and "Annoyed by ducks" )
Untitled Jam Submission A 2D bomberman-like game made in Godot as a submission to a school game jam.
As the unaware player lays bombs to destroy terrain, they need to learn the effect of each block type the hard way to reach the end of the maze in one piece. The effects are randomly assigned at the start of a new game, mean each try is unique.
I was in charge of implementing random effects for each type of block as well as terrain generation based on a hard-coded ASCII description.
Jam Theme: "It's not supposed to do that!"
